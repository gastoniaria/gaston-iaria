Proyectos

Home Banking

Uno de mis proyectos destacados es un sistema desarrollado con Node.js, Express, MongoDB. La aplicación permite a los usuarios realizar transferencias, consultar movimientos, comprar dólares, y administrar cuentas bancarias con alta seguridad.

El sistema cuenta con endpoints REST para cada operación financiera, validaciones robustas con middleware, manejo de sesiones con JWT, y una arquitectura modular para facilitar mantenimiento y escalabilidad.

También incluye notificaciones y registro detallado de transacciones, buscando brindar una experiencia de usuario eficiente y confiable.

Podés ver el código y más detalles en mi repositorio de GitHub: https://github.com/gastoniaria/ProbandoJwtMvc
